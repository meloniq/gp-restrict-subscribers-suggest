GlotPress plugin to restrict subscribers from suggesting translations.

== Description ==

GlotPress plugin to restrict subscribers from suggesting translations.

In case a subscriber has extra permissions, being a validator for a specific locale,
they will be able to add translations to that locale, but not to others.


= Configuration =

Once you have installed GP Restrict Subscribers Suggest it is ready to use.
